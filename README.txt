

STEP BY STEP INSTRUCTIONS:
	1. Insert answers into Answers Folder
	2. Unzip if the files are zipped (Example is included in the folder)
	3. Make sure that filenames in Question folder match the folder names in Answers Folder
	4. Make sure that student.xlsx file exists in Student Folder
	5. Run the codes below:


TO RUN:
	cd `PATH_TO_FOLDER` (CHECK THIS IN PROPERTIES)
	pip install pandas (IF RUNNING THIS FOR THE FIRST TIME)
	python ans_reader.py

NOTES:
	Students Folder:
		Lists all Student Numbers
		If there is an updated list of students, put in Students folder and rename as students.xlsx

	Answers Folder:
		Put folders of Student Answers (Make sure to extract zip files, script will only read folders)

	Questions Folder:
		Make sure that question files are named the same as answer folders in Answers Folder

	Converted Folder:
		Output excel files
